HTML Basics

HTML is the structure of your webpage. Learn the basics of HTML tags and structure below.

HTML stands for Hyper Text Markup Language.

Basic Structure

An HTML document has a defined structure that includes root, head, and body sections.

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

Elements and Tags

HTML elements are the building blocks of web pages.

Elements are represented by tags.

CSS Basics

CSS styles your HTML structure. Below you'll find an introduction to CSS properties and selectors.

Selectors

Selectors define the elements to which a set of CSS rules apply.

CSS selectors can be simple or complex, based on the needs.

h1 { color: red; font-size: 20px; }

Box Model

Understanding the CSS Box Model is crucial to mastering layout.

div { width: 300px; padding: 10px; border: 1px solid black; margin: 20px; }

CSS is powerful for designing responsive layouts.

JavaScript Basics

JavaScript adds interactivity to your pages. Explore basic syntax and functions.

JavaScript can manipulate both CSS and HTML.

console.log("Hello, world!");

JS is widely used for client-side web development.

APIs

APIs allow communication between software components.

APIs are interfaces for different software programs to interact with each other.

fetch('https://api.example.com')

Rest APIs are commonly used to fetch data from a server.
